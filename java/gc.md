### 垃圾回收器 (Garbage Collector)

垃圾回收器是垃圾回收算法（标记-清除算法、复制算法、标记-整理算法、火车算法）的具体实现。不同商家、不同版本的 JVM 所提供的垃圾收集器可能会有很在差别。

- CMS (Concurrent Mark Sweep)
- G1
- Serial
- ParNew
- Parallel Scavenge
- Serial Old
- Parallel Old
